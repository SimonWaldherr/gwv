# GWV Î²
Golang Web Valve - to be connected to your series of tubes

## install

```go get -u -t simonwaldherr.de/go/gwv```

## features

* HTTP Server
* HTTPS Server
* SPDY Server
* Static File Server
* Automatic SSL cert generator
* Realtime Webserver (SSE)
